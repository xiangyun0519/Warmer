# 暖流 - 产品需求文档 (PRD)

**版本**：v1.0
**日期**：2026-04-13

---

## 1. Product Overview

暖流是一款面向大学生的关系维护工具（单机版MVP），帮助用户记录、维护重要的人际关系，让每一段关系都被看见。

- 解决大学生关系疏远却不自知的痛点，提供温暖治愈的用户体验
- 目标用户：18-25岁大学生群体

---

## 2. Core Features

### 2.1 Feature Module
1. **首页**: 关系卡片展示、暖心建议、添加入口
2. **关系详情页**: 温度曲线、记录历史、关系管理
3. **添加/编辑页**: 关系信息录入

### 2.2 Page Details

| Page Name | Module Name | Feature description |
|-----------|-------------|---------------------|
| 首页 | 关系卡片区域 | 按红黄绿分组展示关系卡片，显示最后联系时间 |
| 首页 | 暖心建议区域 | 展示针对性的关系维护建议 |
| 首页 | 添加按钮 | 点击进入添加关系页面 |
| 关系详情页 | 温度曲线 | 使用折线图可视化关系温度变化 |
| 关系详情页 | 温度记录 | 历史记录列表，支持添加新记录 |
| 添加/编辑页 | 关系表单 | 填写昵称、选择头像、关系类型、等级、生日 |

---

## 3. Core Process

```
用户打开App
    ↓
看到"关系温度首页"（按红黄绿排序的所有关系卡片）
    ↓
点击"+"添加一个人
    ↓
填写昵称、选择头像、选关系类型
    ↓
添加成功，回到首页
    ↓
点击某人的卡片
    ↓
看到关系详情页（温度曲线、记录、建议）
    ↓
每周回来更新温度 → 看到新的暖心建议
```

## 4. User Interface Design

### 4.1 Design Style
- **Primary Color**: 暖橙色 (#FF9F43)
- **Secondary Color**: 暖白色 (#FFF9F0)
- **Button Style**: 圆角按钮 + 柔和阴影
- **Fonts**: 优雅的无衬线字体，标题使用柔和的手写感字体
- **Layout Style**: 卡片式布局，大量圆角元素
- **Icon Style**: Emoji图标，温暖治愈风格

### 4.2 Page Design Overview

| Page Name | Module Name | UI Elements |
|-----------|-------------|-------------|
| 首页 | Hero区域 | 暖橙色渐变背景，品牌名称和slogan |
| 首页 | 关系分组 | 🔴需要关心、🟡保持联系、🟢关系健康，每组展示卡片列表 |
| 关系详情页 | 温度曲线 | 渐变折线图，突出温度变化趋势 |
| 添加页 | 表单 | 柔和的输入框，emoji选择器，直观的选项 |

### 4.3 Responsiveness
- 移动端优先设计
- 自适应不同屏幕尺寸
- 触摸优化的交互

## 5. 暖心建议规则

| 场景 | 建议文案 |
|------|----------|
| 14天没联系+降温 | "你和{name}已经两周没联系了，发个消息吧" |
| 7天没联系 | "记得和{name}保持联系哦" |
| 持续降温 | "关系需要维护，找个机会聊聊？" |
| 持续升温 | "你们最近关系升温很快，可以约出来聚聚" |
| 新关系 | "刚认识{name}，可以主动打个招呼" |
| 生日当天 | "今天是{name}的生日，送个祝福吧" |
| 恋人+长期未更新 | "和{name}在一起很久了，记得制造小惊喜" |
| 家人+长期未更新 | "给家人打个电话吧，他们在想你" |
| 温度低于40 | "这段关系好像遇到了问题，也许需要主动修复" |
| 温度高于80 | "这段关系很健康，继续保持" |
